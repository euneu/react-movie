# Nflex

영화 추천 앱!
사용자에게 카테고리 별로 또는 인기 있는 영화를 추천합니다

![image](https://user-images.githubusercontent.com/41333676/179671907-f9fa41af-7de2-4fad-a923-ce794a356192.png)


Link : [둘러보기](https://euneu.github.io/react-movie/)


## file explaination

#### ► atom

- NavList : 전역 변수 설정

#### ➤ component

- Navbar : 상단 메뉴바

- Slide : 메인 화면에서 보여지는 슬라이더 화면

- MovieSlide : 슬라이더 화면의 각 영화 카드

- Movie : 장르별 인기 있는 영화 목록


#### ➤ router

모든 경로 코드가 있습니다 

가져온 영화 api 정보가 들어있고 컴포넌트로 뿌려줍니다
